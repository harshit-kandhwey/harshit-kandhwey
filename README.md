## 🚀 Featured Projects

### [Ephemeral Deploy — Production AWS/ECS DevOps Pipeline](https://github.com/harshit-kandhwey/ephemeral-deploy)

> Production-grade containerised REST API deployment on AWS with full observability and keyless CI/CD

- Modular Terraform across **8 modules** (VPC, ECS, RDS, ElastiCache, ECR, IAM, security groups, monitoring)
- **OIDC-based CI/CD** — zero stored credentials, blue-green deployments with SSM active-slot switching
- Layered secrets pipeline: SSM KMS-encrypted → Secrets Manager → ECS runtime injection
- Dual observability: **Prometheus + Grafana + CloudWatch** dashboards
- Fargate Spot dev environments with 30-min TTL — cost per run: **~$0.02**

`AWS` `ECS Fargate` `Terraform` `GitHub Actions` `Docker` `Prometheus` `Grafana`
